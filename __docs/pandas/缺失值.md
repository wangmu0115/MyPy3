识别数据表或 DataFrame 中缺失值的方法一般分为两种：通过一个覆盖全局的**掩码 mask**表示缺失值；通过一个**哨兵值 sentinel value**表示缺失值。

pandas 中两种存储和处理空值的“模式”：

1. 默认使用哨兵值的缺失数据方案，哨兵值为`NaN`或`None`，取决于数据类型。
2. 使用 pandas 提供的可空值数据类型，这样会创建一个附带的掩码数据来跟踪缺失数据，这些缺失数据会以特殊的`pd.NA`值呈现。



None 哨兵值
任何包含 None 的数组类型会被设置为dtype=object，表示这个数组是由Python对象构成的，在常见的快捷操作时，会消耗更多的资源。
由于Python不支持None的算术运算，使用类似sum, max 这样的聚合运算时会导致出错。
pandas数值数组中不使用None作为哨兵值

NaN作为数值数组中缺失值的哨兵值

math.nan, np.nan

无论与NaN进行任何操作最终结果都是 NaN
NaN的缺点是，它只是一个浮点数值，对于整数、字符串或其他数据类型，没有等效的 NaN 值。

NaN 和 None
虽然 NaN 和 None 有不同的用处，不过 pandas 将他们看成是可以等价交换的，在适当的时候会将两者进行替换。
